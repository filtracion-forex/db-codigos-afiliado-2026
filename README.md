¿Cuantas veces has visto un codigo de afiliado en internet y no sabes si realmente funciona? Por eso empece a probar cada codigo personalmente. Deposito dinero real, espero el bono, y document el resultado.

═ DATOS CUBIERTOS ═

El archivo `codigos.json` contiene para cada broker:

- Codigo exacto
- Bono maximo verificable (no el que anuncian)
- Deposito minimo real
- Si tiene bono sin deposito
- Estado actual de mi ultima prueba
- Notas sobre cada prueba

═ BROKERS ANALIZADOS ═

AvaTrade, XM, Exness, AvaFutures, Pepperstone, IC Markets, eToro, Plus500

═ RESULTADO ═

De los 8 brokers que analice, solo 3 tienen codigos que realmente funcionan con bonos verificables depositando dinero real. Los otros 5 o no tienen programa de codigos publicos o los bonos tienen tantas restricciones que casi nadie los cobra.

═ METODOLOGIA ═

1. Busco el codigo en fuentes publicas
2. Lo pruebo con un deposito real
3. Espero a que el bono se acredite
4. Documento el resultado con captura

Si tienes un codigo que no aparece aqui y quieres que lo verifique, abre un issue.

═ ADVERTENCIA ═

Los brokers cambian sus condiciones sin avisar. Un codigo que funciona hoy puede dejar de funcionar manana.
